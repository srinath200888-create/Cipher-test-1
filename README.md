# Cipher Knowledge

The ultimate OpenCode power-up pack. Drop this into any project and instantly supercharge your AI coding agent with custom agents, domain skills, power commands, MCP servers, and a golden index of the best GitHub repositories.

## What You Get

- **19 Custom Agents** — architect, builder, debugger, ui-designer, vibe-coder, and more
- **14 Domain Skills** — Tauri desktop, React + shadcn, full-stack web, AI/ML, mobile, CLI tools
- **16 Power Commands** — `/new-desktop-app`, `/ship`, `/review`, `/smell`, `/suggest-stack`
- **6 Pre-Configured MCP Servers** — GitHub, Playwright, Filesystem, Postgres, Sentry, Figma
- **Golden Repo Index** — 15 categories of the best open-source tools, researched and ranked
- **Power Prompts** — Drop-in blueprints for building any type of application
- **Architecture & Design Concepts** — Clean architecture, SOLID, design patterns, security

## Quick Start

```powershell
# Clone into your project
cd your-project
git clone https://github.com/srinath200888/cipher-knowledge

# Run setup (creates symlinks and copies config)
.\cipher-knowledge\scripts\setup.ps1

# Restart OpenCode
opencode
```

| Command | Description |
|---------|-------------|
| `/new-desktop-app` | Scaffold a Tauri v2 + React 19 + shadcn/ui desktop app |

| `/new-mobile-app` | Scaffold an Expo/React Native or Flutter mobile app |

| `/deploy` | Deploy to production |
| `/smell` | Detect code smells |

| `/learn` | Add a repo to the index |
| `/suggest-stack` | Recommend a tech stack |
