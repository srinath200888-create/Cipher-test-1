# DevOps & Deployment

Infrastructure, CI/CD, and deployment tools.

## 🔥 Docker
- **Repo**: docker/docker (moby/moby)
- **Stars**: 69K+
- **License**: Apache 2.0
- **Why**: Containerization standard. Package apps with all dependencies, run anywhere.
- **Install**: `winget install Docker.DockerDesktop`
- **Use**: All project deployment

## 🔥 GitHub Actions
- **Repo**: actions/runner
- **Stars**: 5K+
- **License**: MIT
- **Why**: Built-in CI/CD for GitHub repos. Matrix builds, cache, secrets, 2000 free minutes/month.
- **Use**: CI/CD for all projects

## ⭐ Railway
- **Stars**: N/A (proprietary platform, not a repo)
- **Why**: Simplest deployment platform. Connect GitHub, auto-deploys, free tier includes PostgreSQL. One-click deploys.
- **Use**: Full-stack web app deployment

## ⭐ Fly.io
- **Stars**: N/A (proprietary platform)
- **Why**: Global edge deployment. Run Docker containers close to users, generous free tier, automatic HTTPS.
- **Use**: Dockerized app deployment

## ⭐ Vercel
- **Stars**: N/A (proprietary platform)
- **Why**: Best for Next.js. Edge functions, ISR, analytics, auto-SSL. Free for personal projects.
- **Use**: Next.js frontend deployment

## ⭐ Terraform
- **Repo**: hashicorp/terraform
- **Stars**: 43K+
- **Language**: Go
- **License**: MPL 2.0
- **Why**: Infrastructure as Code. Define cloud resources in config files. Works with AWS, Azure, GCP, etc.
- **Install**: `winget install hashicorp.terraform`
- **Use**: Infrastructure management
