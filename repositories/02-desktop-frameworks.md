# Desktop Application Frameworks

Build native desktop applications that users can download and run.

## 🔥 Tauri v2
- **Repo**: tauri-apps/tauri
- **Stars**: 85K+
- **Language**: Rust + Web
- **License**: MIT
- **Why**: Tiny binaries (3-5MB vs Electron's 100MB+), secure by design, cross-platform. The #1 choice for new desktop apps in 2026. Rust backend, any web frontend.
- **Install**: `npm create tauri-app@latest`
- **Use**: All new desktop apps

## ⭐ Nucleus (Kotlin Desktop)
- **Repo**: NucleusFramework/Nucleus
- **Stars**: Rapidly growing
- **Language**: Kotlin + Compose Multiplatform
- **License**: Apache 2.0
- **Why**: Native JVM desktop. 0.5s cold start, 100-150MB RAM, JetBrains-grade UI with Jewel framework. GraalVM native image support.
- **Install**: Via Gradle plugin
- **Use**: Kotlin/JVM teams, performance-critical apps

## ⭐ GTKX
- **Repo**: gtkx-org/gtkx
- **Stars**: 293
- **Language**: TypeScript + Rust
- **License**: MPL 2.0
- **Why**: Build native Linux apps with React + GTK4. Full GTK4 API bindings, HMR, TypeScript-first.
- **Install**: `npm create gtkx-app`
- **Use**: Linux-native apps with React

## 📦 Electron
- **Repo**: electron/electron
- **Stars**: 115K+
- **Language**: JavaScript + C++
- **License**: MIT
- **Why**: Mature, massive ecosystem, powers VS Code and Slack. Heavy (100MB+), but unmatched library support.
- **Install**: `npm install electron`
- **Use**: When you need Electron's ecosystem (Chrome DevTools, etc.)

## 🔥 Tauri Desktop Starter
- **Repo**: DunHouGo/tauri-desktop-starter
- **Stars**: Active
- **Language**: TypeScript + Rust
- **License**: MIT
- **Why**: Production-ready Tauri v2 + React 19 template. Includes shadcn/ui, Zustand, TanStack Query, SQLite, i18n, auto-updates, system tray.
- **Clone**: `git clone` the template
- **Use**: Starting new Tauri projects
