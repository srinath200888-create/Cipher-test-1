# Backend Frameworks

Server-side frameworks for building APIs and services.

## 🔥 FastAPI
- **Repo**: fastapi/fastapi
- **Stars**: 80K+
- **Language**: Python
- **License**: MIT
- **Why**: Modern, fast Python web framework. Async support, automatic OpenAPI docs, Pydantic validation. Best for building APIs in 2026.
- **Install**: `pip install fastapi uvicorn`
- **Use**: Python API development

## 🔥 Next.js
- **Repo**: vercel/next.js
- **Stars**: 128K+
- **Language**: TypeScript
- **License**: MIT
- **Why**: React framework with SSR, SSG, API routes, App Router. Can serve as both frontend and backend.
- **Install**: `npx create-next-app@latest`
- **Use**: React full-stack apps

## ⭐ NestJS
- **Repo**: nestjs/nest
- **Stars**: 68K+
- **Language**: TypeScript
- **License**: MIT
- **Why**: Enterprise-grade Node.js framework. Decorators, DI, modular architecture, GraphQL + REST support.
- **Install**: `npm i -g @nestjs/cli && nest new project`
- **Use**: Enterprise TypeScript APIs

## ⭐ Django
- **Repo**: django/django
- **Stars**: 81K+
- **Language**: Python
- **License**: BSD
- **Why**: Full-featured Python web framework. Admin panel, ORM, auth, middleware. Best for content-heavy apps.
- **Install**: `pip install django`
- **Use**: Content-heavy web apps

## ⭐ Express
- **Repo**: expressjs/express
- **Stars**: 66K+
- **Language**: JavaScript
- **License**: MIT
- **Why**: Minimal Node.js framework. The foundation of the Node.js ecosystem. Lightweight, flexible.
- **Install**: `npm install express`
- **Use**: Simple Node.js APIs

## ⭐ Axum
- **Repo**: tokio-rs/axum
- **Stars**: 21K+
- **Language**: Rust
- **License**: MIT
- **Why**: Ergonomic Rust web framework built on Tokio. Type-safe, fast, great for high-performance APIs.
- **Install**: Add to Cargo.toml
- **Use**: High-performance Rust APIs
