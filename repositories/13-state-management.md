# State Management

Libraries for managing application state.

## 🔥 Zustand
- **Repo**: pmndrs/zustand
- **Stars**: 50K+
- **Language**: TypeScript
- **License**: MIT
- **Why**: Minimal, fast, scalable state management. No boilerplate, no providers, works outside React. The default choice in 2026.
- **Install**: `npm install zustand`
- **Use**: Global state in React/Tauri apps

## 🔥 TanStack Query
- **Repo**: TanStack/query
- **Stars**: 43K+
- **Language**: TypeScript
- **License**: MIT
- **Why**: Server state management. Caching, background refetch, optimistic updates, infinite queries. Replaces Redux for server data.
- **Install**: `npm install @tanstack/react-query`
- **Use**: Server data fetching and caching

## ⭐ Jotai
- **Repo**: pmndrs/jotai
- **Stars**: 19K+
- **Language**: TypeScript
- **License**: MIT
- **Why**: Atomic state management. Like Zustand but with atom primitives. Good for fine-grained reactivity.
- **Install**: `npm install jotai`
- **Use**: Fine-grained reactive state

## ⭐ Redux Toolkit
- **Repo**: reduxjs/redux-toolkit
- **Stars**: 11K+
- **Language**: TypeScript
- **License**: MIT
- **Why**: The official Redux toolkit. Simplified Redux with createSlice, createAsyncThunk, RTK Query.
- **Install**: `npm install @reduxjs/toolkit react-redux`
- **Use**: Large apps needing Redux ecosystem
