# AI Coding Agents

Tools that use AI to write, review, and manage code.

## 🔥 OpenCode
- **Repo**: anomalyco/opencode
- **Stars**: 180K+
- **Language**: TypeScript
- **License**: MIT
- **Why**: Open-source AI coding agent you're using right now. Terminal TUI + desktop app. Built-in build/plan agents. Extensible with skills, plugins, MCP.
- **Install**: `npm i -g opencode-ai@latest`
- **Use**: Daily driver for AI-assisted coding

## 🔥 OpenAI Codex
- **Repo**: openai/codex
- **Stars**: 94K+
- **Language**: Rust
- **License**: Apache 2.0
- **Why**: OpenAI's official coding agent. Fast, CLI-native, deeply integrated with ChatGPT.
- **Install**: `npm install -g @openai/codex`
- **Use**: When you prefer OpenAI models

## 🔥 Claude Code
- **Repo**: anthropics/claude-code (via Anthropic)
- **Stars**: 134K+
- **Language**: TypeScript
- **License**: Proprietary
- **Why**: Anthropic's terminal coding agent. Excellent at long context, complex reasoning.
- **Install**: Via Anthropic console
- **Use**: Complex reasoning tasks

## ⭐ OpenAI Symphony
- **Repo**: openai/symphony
- **Stars**: 25K+
- **Language**: Elixir
- **License**: Apache 2.0

## ⭐ GitAgent
- **Repo**: open-gitagent/gitagent
- **Stars**: 580+
- **Language**: TypeScript
- **License**: MIT
- **Why**: Git-native agent framework. Agent IS a git repo — identity, rules, memory all version-controlled.
- **Install**: `npm install -g @open-gitagent/gitagent`
- **Use**: Version-controlled agent configs
