# Design Resources

Icons, fonts, design kits, and visual resources.

## 🔥 Lucide Icons
- **Repo**: lucide-icons/lucide
- **Stars**: 14K+
- **License**: MIT
- **Why**: Beautiful, consistent open-source icons. 1500+ icons, tree-shakeable React package. Used by shadcn/ui by default.
- **Install**: `npm install lucide-react`
- **Use**: Icons in any project

## ⭐ Heroicons
- **Repo**: tailwindlabs/heroicons
- **Stars**: 22K+
- **License**: MIT
- **Why**: Beautiful hand-crafted SVG icons by Tailwind CSS team. Outline and solid variants.
- **Install**: `npm install @heroicons/react`
- **Use**: Web app icons

## ⭐ Radix Icons
- **Repo**: radix-ui/icons
- **Stars**: 2K+
- **License**: MIT
- **Why**: Icon set from the Radix UI team. 15x15 pixel grid, consistent with Radix components.
- **Install**: `npm install @radix-ui/react-icons`
- **Use**: Radix UI projects

## ⭐ Geist Font
- **Repo**: vercel/geist-font
- **Stars**: 2K+
- **License**: Custom (free)
- **Why**: Vercel's typeface, used by shadcn/ui by default. Clean, modern sans-serif + mono.
- **Install**: `npm install geist`
- **Use**: Modern app typography

## ⭐ Untitled UI
- **Repo**: untitled-ui (figma.com/community)
- **License**: Paid (Figma)
- **Why**: Professional UI kit for Figma. 4000+ components, 700+ styles. Used by many production teams.
- **Use**: Figma design systems
