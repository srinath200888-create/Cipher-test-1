# Mobile Development

Frameworks for building iOS and Android applications.

## 🔥 Expo + React Native
- **Repo**: expo/expo
- **Stars**: 36K+
- **Language**: TypeScript
- **License**: MIT
- **Why**: The most popular way to build React Native apps. File-based routing (Expo Router), OTA updates, EAS Build for CI/CD, 50+ native modules.
- **Install**: `npx create-expo-app@latest`
- **Use**: Cross-platform mobile apps

## ⭐ Flutter
- **Repo**: flutter/flutter
- **Stars**: 168K+
- **Language**: Dart
- **License**: BSD
- **Why**: Google's UI toolkit for natively compiled mobile apps. Best for custom UI, high-performance graphics, pixel-perfect cross-platform.
- **Install**: `winget install flutter`, `flutter create my_app`
- **Use**: Custom UI, high-performance graphics

## ⭐ Capacitor
- **Repo**: ionic-team/capacitor
- **Stars**: 12K+
- **Language**: TypeScript
- **License**: MIT
- **Why**: Turn any web app into a native mobile app. Used by Keel template. WebView-based with native API access.
- **Install**: `npm install @capacitor/core @capacitor/cli`
- **Use**: Web-to-mobile conversion

## 🔥 React Native
- **Repo**: facebook/react-native
- **Stars**: 120K+
- **Language**: TypeScript/C++
- **License**: MIT
- **Why**: The original cross-platform mobile framework. New Architecture (Fabric, TurboModules) brings major perf improvements.
- **Install**: `npx react-native init MyApp`
- **Use**: When you need native modules not in Expo
