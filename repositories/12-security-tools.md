# Security Tools

Tools for securing applications and infrastructure.

## ⭐ OWASP ZAP
- **Repo**: zaproxy/zaproxy
- **Stars**: 13K+
- **Language**: Java
- **License**: Apache 2.0
- **Why**: Full-featured web app security scanner. Automated scanning, passive/active scanning, API-based testing.
- **Install**: `winget install OWASP.ZAP`
- **Use**: Web app security scanning

## ⭐ Semgrep
- **Repo**: semgrep/semgrep
- **Stars**: 11K+
- **Language**: OCaml/Python
- **License**: LGPL 2.1
- **Why**: Static code analysis. Custom rules, OWASP Top 10 coverage, supports 20+ languages.
- **Install**: `pip install semgrep`
- **Use**: SAST, custom security rules

## ⭐ Trivy
- **Repo**: aquasecurity/trivy
- **Stars**: 24K+
- **Language**: Go
- **License**: Apache 2.0
- **Why**: Vulnerability scanner for containers, dependencies, and code. Fast, comprehensive CVE database.
- **Install**: `winget install aquasecurity.Trivy`
- **Use**: Container and dependency scanning

## ⭐ gitleaks
- **Repo**: gitleaks/gitleaks
- **Stars**: 18K+
- **Language**: Go
- **License**: MIT
- **Why**: Secrets scanner for git repos. Detects hardcoded API keys, passwords, tokens before commits.
- **Install**: `winget install gitleaks`
- **Use**: Secrets detection in repos

## ⭐ SonarQube
- **Repo**: SonarSource/sonarqube
- **Stars**: 9K+
- **Language**: Java
- **License**: LGPL 3.0
- **Why**: Continuous code quality inspection. Bugs, vulnerabilities, code smells, technical debt tracking.
- **Install**: Docker or local server
- **Use**: Enterprise code quality
