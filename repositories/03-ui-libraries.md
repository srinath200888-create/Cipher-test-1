# UI Libraries & Component Collections

Beautiful, accessible UI components for your apps.

## 🔥 shadcn/ui
- **Repo**: shadcn-ui/ui
- **Stars**: 117K+
- **Language**: TypeScript
- **License**: MIT
- **Why**: The standard for React UI in 2026. Copy-paste components you own entirely. Built on Radix UI + Tailwind CSS. AI-ready open code. 76+ components.
- **Install**: `npx shadcn@latest add button`
- **Use**: Every React project

## 🔥 Radix UI
- **Repo**: radix-ui/primitives
- **Stars**: 16K+
- **Language**: TypeScript
- **License**: MIT
- **Why**: Headless, accessible UI primitives that shadcn/ui is built on. Dialog, DropdownMenu, Popover, Tooltip, etc. Full WAI-ARIA compliance.
- **Install**: `npm install @radix-ui/react-dialog`
- **Use**: Building custom accessible components

## 🔥 Tailwind CSS v4
- **Repo**: tailwindlabs/tailwindcss
- **Stars**: 84K+
- **Language**: CSS/TypeScript
- **License**: MIT
- **Why**: Utility-first CSS framework. The industry standard for styling in 2026. v4 is faster, smaller, CSS-first config.
- **Install**: `npm install tailwindcss @tailwindcss/vite`
- **Use**: Every web/desktop project

## ⭐ Framer Motion
- **Repo**: framer/motion
- **Stars**: 24K+
- **Language**: TypeScript
- **License**: MIT
- **Why**: Declarative React animation library. Layout animations, gesture support, spring physics. Used by shadcn/ui for animations.
- **Install**: `npm install framer-motion`
- **Use**: Adding animations

## ⭐ Lucide Icons
- **Repo**: lucide-icons/lucide
- **Stars**: 14K+
- **Language**: TypeScript/SVG
- **License**: MIT
- **Why**: Beautiful, consistent open-source icons. Used by shadcn/ui by default. Tree-shakeable, 1500+ icons.
- **Install**: `npm install lucide-react`
- **Use**: Icons in any project

## ⭐ MUI (Material UI)
- **Repo**: mui/material-ui
- **Stars**: 94K+
- **Language**: TypeScript
- **License**: MIT
- **Why**: Mature, comprehensive component library. Best for enterprise apps needing Material Design out of the box.
- **Install**: `npm install @mui/material @emotion/react`
- **Use**: Enterprise apps, Material Design
