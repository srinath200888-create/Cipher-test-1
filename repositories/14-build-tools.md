# Build Tools

Bundlers, compilers, and toolchains.

## 🔥 Vite
- **Repo**: vitejs/vite
- **Stars**: 71K+
- **Language**: TypeScript
- **License**: MIT
- **Why**: Next-gen build tool. Lightning-fast HMR, native ESM dev server, Rollup-based production builds. The standard in 2026.
- **Install**: `npm create vite@latest`
- **Use**: Frontend bundling

## ⭐ Bun
- **Repo**: oven-sh/bun
- **Stars**: 75K+
- **Language**: Zig
- **License**: MIT
- **Why**: All-in-one JavaScript runtime, bundler, test runner, package manager. Fast startup, Node.js compatible.
- **Install**: `winget install Bun.Bun`
- **Use**: JavaScript development

## ⭐ Turbopack
- **Repo**: vercel/turbopack
- **Stars**: 26K+
- **Language**: Rust
- **License**: MIT
- **Why**: Incremental bundler by Vercel. Used by Next.js, Rust-based, fast for large projects.
- **Use**: Next.js projects (included)

## ⭐ Cargo
- **Repo**: rust-lang/cargo
- **Stars**: N/A (Rust ships with Cargo)
- **Why**: Rust's build system and package manager. Build, test, bench, doc, publish — all in one tool.
- **Use**: Rust project builds

## ⭐ esbuild
- **Repo**: evanw/esbuild
- **Stars**: 38K+
- **Language**: Go
- **License**: MIT
- **Why**: Extremely fast JavaScript bundler and minifier. Used by Vite internally for production builds.
- **Install**: `npm install esbuild`
- **Use**: Fast JS/TS bundling
