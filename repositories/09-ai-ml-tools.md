# AI/ML Tools

Libraries and tools for building AI-powered applications.

## 🔥 LangChain
- **Repo**: langchain-ai/langchain
- **Stars**: 100K+
- **Language**: Python
- **License**: MIT
- **Why**: The standard framework for LLM-powered applications. Chains, agents, RAG, tool calling, streaming.
- **Install**: `pip install langchain langchain-community langchain-openai`
- **Use**: LLM application development

## ⭐ LlamaIndex
- **Repo**: run-llama/llama_index
- **Stars**: 38K+
- **Language**: Python
- **License**: MIT
- **Why**: Data framework for LLM apps. Excel at RAG, document indexing, structured data retrieval.
- **Install**: `pip install llama-index`
- **Use**: RAG and document Q&A

## 🔥 Ollama
- **Repo**: ollama/ollama
- **Stars**: 120K+
- **Language**: Go
- **License**: MIT
- **Why**: Run LLMs locally with one command. Supports Llama 3.2, Mistral, Phi, Gemma. OpenAI-compatible API.
- **Install**: `winget install Ollama.Ollama`
- **Use**: Local LLM inference

## 🔥 ChromaDB
- **Repo**: chroma-core/chroma
- **Stars**: 17K+
- **Language**: Python/Rust
- **License**: Apache 2.0
- **Why**: The AI-native open-source embedding database. Easy setup, good for prototyping and production.
- **Install**: `pip install chromadb`
- **Use**: Vector storage for RAG

## ⭐ Qdrant
- **Repo**: qdrant/qdrant
- **Stars**: 22K+
- **Language**: Rust
- **License**: Apache 2.0
- **Why**: High-performance vector database. Rust-backed, fast filters, rich API, self-host or cloud.
- **Install**: `docker run qdrant/qdrant`
- **Use**: Production vector search

## ⭐ Mem0
- **Repo**: mem0ai/mem0
- **Stars**: 52K+
- **Language**: Python
- **License**: Apache 2.0
- **Why**: Memory layer for AI agents. Persistent context, entity extraction, long-term memory across sessions.
- **Install**: `pip install mem0ai`
- **Use**: AI agent memory

## ⭐ LocalAI
- **Repo**: mudler/LocalAI
- **Stars**: 44K+
- **Language**: Go
- **License**: MIT
- **Why**: OpenAI API-compatible local inference. Run models on CPU/GPU, no GPU required for smaller models.
- **Install**: `docker run localai/localai`
- **Use**: Local OpenAI-compatible API
