# Build a Mobile App

Use this prompt when you want to build a complete mobile application.

## Prompt

Build a complete mobile application for [describe your app].

### Requirements
- **Platform**: iOS + Android
- **Stack**: Expo + React Native + TypeScript
- **Navigation**: Expo Router (file-based)
- **State**: Zustand + TanStack Query
- **Submission**: App Store + Google Play

### Process
1. Plan the app structure and screens
2. Design the data flow
3. Scaffold with create-expo-app
4. Set up Expo Router (tabs, stack navigation)
5. Implement screens and components
6. Add native APIs (camera, location, notifications as needed)
7. Set up secure storage for auth tokens
8. Implement offline support
9. Test on iOS and Android simulators
10. Build for production (EAS Build)

### Deliverables
- Working mobile app on iOS and Android
- File-based navigation with Expo Router
- Native API integrations
- EAS Build configuration for store submission
